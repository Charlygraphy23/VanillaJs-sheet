# VanillaJs-sheet
> This is a demo of google sheet. Functionalites are - 
 - Add data to sheet
 - UI select multiple rows
 - upload data as CSV format
 - download sheet data as csv format

#Demo - [here](https://astonishing-bavarois-450233.netlify.app/)

# Stack 
- Vanilla JS
- Css
- Html
